LAME Encoder Information

Max uses the LAME encoding engine to generate MP3 files. LAME is an high-quality, open-source encoding engine.

The following table was taken from an HydrogenAudio forum post regarding LAME quality at various settings.

# Recommended encoder settings:

The settings, presets are listed beginning with highest 
possible quality (-b 320) stepping down to lower qualities:

CODE
switch           equals preset        target kbit/s   bitrate range kbit/s

-b 320           = --preset insane          320        320 CBR
-V 0 --vbr-new   = --preset fast extreme    245        220-260
-V 0             = --preset extreme         245        220-260
-V 1 --vbr-new                              225        200-250
-V 1                                        225        200-250
-V 2 --vbr-new   = --preset fast standard   190        170-210
-V 2             = --preset standard        190        170-210
-V 3 --vbr-new                              175        155-195
-V 3                                        175        155-195
-V 4 --vbr-new   = --preset fast medium     165        145-185
-V 4             = --preset medium          165        145-185
-V 5 --vbr-new                              130        110-150
-V 5                                        130        110-150
-V 6 --vbr-new                              115         95-135
-V 6                                        115         95-135
-V 7 --vbr-new                              100         80-120
-V 7                                        100         80-120
